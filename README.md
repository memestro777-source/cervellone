🧠 Big Brain!
A fast-paced brain-training game. Test yourself across four categories — Visual, Logic, Math, and Memory — earn a brain size in cm³, and beat your record. Single HTML file, zero dependencies, works offline.
Game Modes

🎲 Random challenge — 4 stages of 45 seconds, one surprise minigame per category
🎯 Custom challenge — same 4 stages, but you pick every game
🎮 Single game — practice any of the 16 minigames on its own, with a personal best saved per game

The 16 Minigames

👁️ Visual: Cube Counter, Missing Piece, Bubbles, Odd One Out
🧩 Logic: The Gears, Hexagon Path, The Balance, Symbol Sudoku
➗ Math: Missing Operators, The Number Plate, Clock Difference, Broken Equality
💭 Memory: Flip & Find, Shape Order, Action Sequence, Memory Grid

Features

Invisible adaptive difficulty: 3 correct answers step the level up, repeated mistakes step it back down — no points ever lost on errors, and the right answer is always revealed
Memory games can be skipped: tap as soon as you've memorized to start answering immediately
Balanced scoring across all games (measured with simulated expert play), with a streak multiplier and speed bonuses
Final animation counting up through every brain type — from 🐣 Hatching Brain to 🌌 Cosmic Mind (2500+) — with confetti for legendary scores
🏆 Persistent records: best brain size, per-game bests, games played
🔊 Fully synthesized sound (Web Audio API), mute toggle
🌍 English/Italian with one-tap flag switch; share buttons for the game link and your Wordle-style score card
📱 Fits any screen with no scrolling in games: boards are sized on the measured play area, from phones to desktop

Running It
Open cervellone.html in any modern browser. No server, no installation needed.
